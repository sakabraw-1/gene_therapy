## üîß TECHNICAL SETUP NOTES

### **Mailchimp Setup:**
1. Create audience/list: "CDKL5 Donors & Supporters"
2. Create segments:
   - One-time donors
   - Monthly recurring donors
   - Event attendees
   - Newsletter subscribers only
3. Set up merge tags: FNAME, AMOUNT, DATE, TOTAL_2025

### **Zapier Workflows:**

**Zap 1: Stripe ‚Üí Thank You Email**
- Trigger: New Stripe payment
- Action: Add to Mailchimp + Send thank you email with amount

**Zap 2: PayPal ‚Üí Thank You Email**
- Trigger: New PayPal payment
- Action: Add to Mailchimp + Send thank you email with amount

**Zap 3: Google Sheet ‚Üí Thank You Email (Manual Donations)**
- Trigger: New row added to "Manual Donations" sheet
- Action: Add to Mailchimp + Send Template 1B (no amount mentioned)
- Note: Use for ALL manual donations (Zelle, Crypto, DAF, Stocks)

**Zap 4: Monthly Recurring Donors**
- Trigger: Stripe subscription payment
- Action: Send monthly thank you email with total-to-date

**Zap 5: Newsletter Signup ‚Üí Confirmation Email**
- Trigger: New subscriber added to Mailchimp (via "Stay Connected" form on website)
- Action: Send newsletter signup confirmation email (Template 3)
- Note: Mailchimp will automatically prevent duplicate subscriptions

### **Google Sheet Template for Manual Donations:**
Columns:
- Date
- Donor First Name
- Donor Last Name
- Email
- Amount
- Payment Method (Zelle/Crypto/DAF/Stocks)
- Notes

**Important:** All manual donations (Zelle, Crypto, DAF, Stocks) use Template 1B which does NOT mention the amount in the email for privacy reasons.

**Summary:**
- **Stripe & PayPal** (automated) = Template 1 (includes amount)
- **Zelle, Crypto, DAF, Stocks** (manual via Google Sheet) = Template 1B (no amount mentioned)
